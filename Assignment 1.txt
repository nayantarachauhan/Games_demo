A and B wants to play the a game.

Game Rules

Both players are given the same string.
Both players have to make substrings using the letters of the string.
Player A has to make words starting with consonants.
Player B has to make words starting with vowels. 
The game ends when both players have made all possible substrings. 

Scoring
The player gets +1 point for each occurrence of the substring in the string .

For Example:
String  = BANANA
Player B's vowel beginning word = ANA
Here, ANA occurs twice in BANANA. Hence, player B will get 2 Points. 


Write a code that takes all said conditions in to account and then prints the winner's name and his score.

Example:

A, 12 points
